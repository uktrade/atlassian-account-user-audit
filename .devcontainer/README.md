# Devconatiner based Development

You must be using VSCode with Dev Container extension and, have docker daemon installed locally to be able to use devcontainer based environment

Additionally, you need to create .env file with minimum of following variables for it to work properly

.env

```bash
ATLASSIAN_AUTH_TOKEN=your_atlassian_token
ATLASSIAN_ORG_NAME=your_atlassian_org_name
ATLASSIAN_URL=https://atlassian.admin.api.url
```
